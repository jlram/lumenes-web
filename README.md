## Project setup
```
npm install

### Instalar Vue Cli
```
npm install -g @vue/cli
```

### Ejecutar cliente
```
vue ui
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
